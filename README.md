# test-task-API-YouTube

### Описание
Приложение для поиска **самых новых** youtube-видео.

### Что использованно
* Ванильный JS
* Bootstrap 4.6
* YouTube Data API v3

Ели видео не ищутся, значит квота на API закончилась. Попропуйте завтра.
